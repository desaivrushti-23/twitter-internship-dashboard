# twitter-internship-dashboard 
Project Description

Twitter Analytics Dashboard using Power BI

Developed an interactive Twitter Analytics Dashboard in Microsoft Power BI to analyze tweet performance and user engagement. The dashboard provides insights into likes, retweets, replies, engagement rate, media interactions, impressions, and top-performing tweets through interactive charts, KPI cards, and slicers. Advanced data preparation, DAX calculations, filtering, and visualization techniques were used to transform raw Twitter data into meaningful business insights. The dashboard enables users to explore engagement trends, compare tweet categories, analyze media performance, and identify high-performing content for better decision-making.

Key Features
Interactive KPI Cards
Tweet Engagement Analysis
Media Performance Analysis
Monthly Engagement Trends
Top 10 Performing Tweets
Dynamic Filters & Slicers
DAX Measures & Calculated Columns
Professional Dashboard Design


Tech Stack
Microsoft Power BI
Power Query
DAX (Data Analysis Expressions)
Microsoft Excel
